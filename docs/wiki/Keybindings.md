# Keybindings

Press `h` or `?` in the app for the same list.

## Looks

One-press visual presets. Start here.

| Key | Action |
| --- | --- |
| `l` | Next Look |
| `L` | Looks gallery, click one to apply |

## Navigation

| Key | Action |
| --- | --- |
| Arrows | Pan |
| Shift + arrow | Pan x4 |
| Mouse drag | Pan smoothly |
| Click | Recentre, or select an aircraft |
| Wheel | Zoom |
| `+` `=` | Zoom in |
| `-` `_` | Zoom out |
| `0`..`9` | Jump to that zoom level |
| `g` | Go to lat,lon |
| `r` | Back to home position |

## Source

| Key | Action |
| --- | --- |
| `k` | Next map style (OSM, Topo, Humanitarian, Positron, DarkMatter, Voyager, Protomaps) |
| `v` | Vector or raster |

## Display

| Key | Action |
| --- | --- |
| `m` | View mode: ascii, quadrant, braille, half |
| `t` | Next theme |
| `p` | Next palette |
| `d` | Next dither (ASCII view only) |
| `s` | Shaded blocks (quadrant and braille only) |
| `c` | Colour on or off |
| `N` | Map labels, meaning place names |
| `u` | Threshold: adaptive, percentile, edge, fixed |

## Image adjust

See [Image adjust](Image-adjust.md) for what these actually do.

| Key | Action |
| --- | --- |
| `[` `]` | Brightness |
| `{` `}` | Contrast |
| `(` `)` | Gamma |
| `<` `>` | Saturation |
| `;` `:` | Black point, lifts a dark theme off black |
| `'` `"` | White point, takes the glare off a light theme |
| `\` | Reset all of the above |

## Aircraft

| Key | Action |
| --- | --- |
| `a` | Labels: smart, all, selected, none |
| `n` | Density: 50, 150, 500, all |
| `G` | Ground traffic on or off |
| `f` | Follow the selected aircraft |

## Sidebar

| Key | Action |
| --- | --- |
| `Tab` | Show or hide |
| `F2` | Move focus between map and sidebar |
| `F3` `F4` | Previous or next tab, works anywhere |
| `F5`..`F8` | Jump to tab 1 to 4 |
| `Ctrl+←` `Ctrl+→` | Previous or next tab, if your terminal sends it |
| `Esc` then arrow | Same, as an Alt+arrow alias |
| `1`..`4` | Jump to tab, when the sidebar has focus |

Tabs are Settings, Search, Controls, Integration, Performance.

## App

| Key | Action |
| --- | --- |
| `w` | Widgets launcher |
| `x` | Save a PNG |
| `Ctrl+S` | Save profile, restored on next start |
| `h` `?` | Help |
| `q` `Ctrl+C` | Quit |
